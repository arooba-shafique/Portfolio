Personal Blog Website

Description

A responsive and interactive personal blog website built using HTML, CSS, JavaScript, and Tailwind CSS. This blog allows users to explore posts by category, search by title, and navigate with pagination. It is visually styled with gradients, shadows, and animations for a modern look.

🛠 Features

  • Category Filter – Browse blog posts by categories like Tech, Travel, and Food.
  • Search Functionality – Search posts dynamically by title.
  • Pagination – Navigate through multiple pages of blog posts easily.
  • Responsive Design – Works on mobile, tablet, and desktop screens.
  • Animated UI – Fade-in animation for blog cards and hover effects for interactive experience.
  • Custom Styling – Tailwind CSS gradients, shadows, and rounded elements for a clean aesthetic.

💻 Tech Stack

  • HTML5 – Structure and layout
  • CSS & Tailwind CSS – Styling, gradients, shadows, responsive layout
  • JavaScript – Dynamic post filtering, search, and pagination
  • Images & Media – Embedded blog images for posts
